About This Project

This is a Django-based job application system that allows users to upload their resumes, extracts relevant information using NLP, and matches their profiles with job postings stored in a MongoDB database.

Setup Instructions

Clone the repository:
git clone 
cd resumeAnalyse
Configure MongoDB connection in settings.py:
DATABASES = {
    'default': {
        'ENGINE': 'djongo',
        'NAME': 'your_database_name',
    }
}
Run migrations:
python manage.py migrate
Start the development server:
python manage.py runserver
Open http://127.0.0.1:8000/ in your browser to access the application.

How It Works

  1. The user fills in the form and uploads a resume.
  2. The resume is parsed using NLP to extract skills, experience, and projects.
  3. The extracted details are compared with job descriptions in MongoDB.
  4. Matching job postings are displayed to the user.
  5. The user can apply if their match percentage is above 60%.

Python Version: Python 3.12.2

Django Version: 5.1.6
